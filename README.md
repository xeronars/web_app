# Weather Reporter
Python script that fetches weather data of cities from API

## Features
- Fetches weather data from Open-Meteo API
- Stores data in PostgreSQL database
- UI for city input and return the weather report of that city

## Requirements
- *Python 3.13.9*
- *PostgreSQL*
